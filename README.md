# 🥚 EggsSharer App (English Description)

**EggsSharer** is a free, cloud-synchronized Progressive Web App (PWA) designed for backyard chicken owners to effortlessly manage production, track customers, and monitor finances.

App URL: [EggsSharerApp](https://schramax.github.io/EggsSharerApp/EggsSharer.html)

---

## ✨ Key Features

- **Dashboard:** Real-time egg stock, sales, and balance overview.
- **Production:** Log daily harvests with automatic duplicate prevention.
- **Sales & CRM:** Manage customers and sales (sales, gifts, personal use).
- **Bancontact Integration:** Android users can share links directly to pre-fill sales/price/eggs.
- **Reports:** Interactive charts with cumulative trends and volume analytics.
- **Multi-device Sync:** Seamlessly share one dataset across multiple family devices.

---

<details>
<summary><b>🛠️ Installation (as a PWA)</b></summary>

**iPhone (iOS — Safari only):**
1. Visit the app URL in **Safari**.
2. Tap the **Share** button → **Add to Home Screen**.

**Android (Chrome):**
1. Visit the app URL in **Chrome**.
2. Tap the **three-dot menu** → **Install app** (or "Add to Home Screen").

Once installed, the app launches in full-screen mode just like a native application.
</details>

<details>
<summary><b>⚙️ Settings & Account</b></summary>

- **Language:** English and French support (app and system emails).
- **Security:** Verified signup, password resets, and email change support.
- **Data Management:** Export/Import JSON backups and "Force Update" capability.
</details>

<details>
<summary><b>☁️ Cloud Sync & Technical Stack</b></summary>

EggsSharer uses **Firebase Authentication** and **Realtime Database** for secure, user-isolated synchronization.

| Layer | Technology |
|---|---|
| Frontend | Vanilla HTML5, CSS3, JavaScript (ES6+) |
| Charts | [Chart.js](https://www.chartjs.org/) |
| Icons | [Lucide Icons](https://lucide.dev/) |
| Cloud Sync | [Firebase](https://firebase.google.com/) |
| Hosting | [GitHub Pages](https://pages.github.com/) |
| Offline | Service Worker (PWA) |
</details>

<details>
<summary><b>🔒 Privacy</b></summary>

All user data is stored exclusively in your private Firebase cloud account, isolated by authenticated user ID. Your flock data is always private.
</details>

# 🥚 EggsSharer App (Description en Français)

**EggsSharer** est une Progressive Web App (PWA) gratuite et synchronisée, conçue pour gérer facilement la production d'œufs, le suivi des clients et les finances de votre poulailler.

URL de l'application : [EggsSharerApp](https://schramax.github.io/EggsSharerApp/EggsSharer.html)

---

## ✨ Points Forts

- **Tableau de bord :** Aperçu du stock, des ventes et du solde en temps réel.
- **Production :** Suivi des récoltes avec gestion automatique des doublons.
- **Ventes & Clients :** Gestion des ventes, dons et usage personnel.
- **Intégration Bancontact :** (Android) Pré-remplissage automatique des ventes via partage.
- **Rapports :** Graphiques interactifs avec tendances cumulées et analyses par volume.
- **Multi-appareil :** Partage d'un seul compte pour toute la famille.

---
